# Linebra

A small library for linear algebra oprations.

## Usage

```ts
// ESM:
import * as linebra from 'linebra';
// CJS:
const linebra = require('linebra');
```

## Defintions

### Vertex

```ts
const v = { x: 1, y: 1 };
```

### Line

```ts
const l = [
  { x: 1, y: 1 },
  { x: 3, y: 3 },
];
```

or

```ts
const l = { m: 1, c: 3 }; // i.e. y = m * x + c
```

### Polygon

An ordered set of vertices:

```ts
const p = [
  { x: 1, y: 1 },
  { x: 3, y: 3 },
  { x: 0, y: 3 },
];
```
